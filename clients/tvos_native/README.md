# `clients/tvos_native`

Native tvOS gameplay client that follows the Python engine contract.

## Contract
- Uses the serialized `GameState` schema from `engine.serialize`.
- Consumes deterministic golden vectors generated by engine tests.
- Maps Siri Remote/controller input to `InputFrame`-equivalent commands.

## Delivery target
- Full gameplay client on tvOS with native rendering/input.
- Python remains source-of-truth for rules and test vectors.

## Build output naming
- `kittychase-tvos-{arch}-{version}.ipa`
