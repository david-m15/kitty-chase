# Legacy Porting Kits

Wave 3 platforms ship with source-first kits and smoke checks before binary packaging.

## Included targets
- FreeBSD
- Solaris
- QNX
- AIX (IBM)
- HP-UX (Hewlett Packard)
- ReactOS
- OpenVMS
- HarmonyOS
- Fuchsia

## Kit contents
- Dependency matrix
- Source build notes
- Platform shim checklist
- Engine headless smoke command
